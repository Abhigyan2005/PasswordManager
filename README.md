# Sentinel
CLI tool to manage your passwords.


## Installation
``` 
npm i -g s3ntin3l
```

## Getting Started
```
sentinel
```

## Features

- CLI based password-manager
- simple workflow
- passwords are saved locally on your system
- encrypted password is stored and it is only decrypted and shown when the user explicitly asks for it.

## status
- this is just a MVP.
- security and better features will be added.
